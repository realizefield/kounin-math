① こんな問題はどうする？
📌 項が多くて公式が使えない…？
高認では こんな形の展開 もよく出題されます：
例)$(x+y+3)(x+y+4)$
一見ややこしいけれど，共通部分 に注目すると簡単に解けます。
よく見ると…
注目
$(\,\underline{x+y}+3\,)(\,\underline{x+y}+4\,)$
発見
両方のかっこに $x+y$ という共通部分がある！
アイデア
$x+y$ を $A$ に置き換えてみよう
② 置き換えのキホン手順
📌 3ステップ
① 共通部分を $A$ と置く
② $A$ を使った式を 展開公式 で展開
③ $A$ を元に戻して すべて展開
例題）$(x+y+3)(x+y+4)$ を展開する
①置き換え
$x+y = A$ とおくと…
②式に
$(A+3)(A+4)$ になる
公式①適用
たして・かける → $A^2 + 7A + 12$
③Aを戻す
$A = x+y$ を代入：$(x+y)^2 + 7(x+y) + 12$
展開する
$(x+y)^2 = x^2 + 2xy + y^2$，$7(x+y) = 7x + 7y$
$x^2 + 2xy + y^2 + 7x + 7y + 12$
💡 ポイント：「共通部分を見つけたら $A$ に置き換え」
かっこの中に 同じカタマリ があれば，それを $A$ と置くチャンス！
ばらしてかけるよりも，ずっと速く・正確に 展開できる。
③ パターン1：$(A+a)(A+b)$ 型
📌 後ろの数が違うパターン
$(x+y+3)(x+y+4)$ や $(x+y-4)(x+y+5)$ のように，
共通部分以外の数が違うときは 公式①（たして・かける） を使う。
例)$(x+y-4)(x+y+5)$ を展開する
置き換え
$x+y = A$ → $(A-4)(A+5)$
公式①
たして $1$，かけて $-20$ → $A^2 + A - 20$
$A$ を戻す
$(x+y)^2 + (x+y) - 20$
展開
$x^2 + 2xy + y^2 + x + y - 20$
$x^2 + 2xy + y^2 + x + y - 20$
④ パターン2：$(A+a)(A-a)$ 型
📌 並び替えて和と差の積に
$(x-3y+1)(x+3y+1)$ のような問題は，一見すると共通部分が見えにくい。
でも 並び替え ると…
$(x+1-3y)(x+1+3y)$
こうなる！　$x+1$ が共通部分で，和と差の積（公式③） が使える！
項を並び替えるときは，符号も一緒に動かす こと！
$-3y$ を後ろに移動するときは，マイナスのまま移動。
例)$(x-3y+1)(x+3y+1)$ を展開する
並び替え
$(x+1-3y)(x+1+3y)$ ← $x+1$ が共通部分
置き換え
$x+1 = A$ → $(A-3y)(A+3y)$
公式③
和と差の積 → $A^2 - 9y^2$
$A$ を戻す
$(x+1)^2 - 9y^2$
展開
$x^2 + 2x + 1 - 9y^2$
$x^2 + 2x + 1 - 9y^2$
⑤ パターン3：$(A+a)^2$ 型
📌 2乗の形になっているパターン
$(x+y-2)^2$ のような問題も，$x+y = A$ と置き換え れば，
$(A-2)^2$ となり 平方公式（公式②） が使える。
例)$(x+y-2)^2$ を展開する
置き換え
$x+y = A$ → $(A-2)^2$
公式②
$A^2 - 4A + 4$
$A$ を戻す
$(x+y)^2 - 4(x+y) + 4$
展開
$x^2 + 2xy + y^2 - 4x - 4y + 4$
$x^2 + 2xy + y^2 - 4x - 4y + 4$
⑥ 3つのパターンを比べてみよう
パターン1：$(A+a)(A+b)$ 型 → 公式①
例)$(x+y+3)(x+y+4)$
→ $(A+3)(A+4) = A^2 + 7A + 12$
パターン2：$(A+a)(A-a)$ 型 → 公式③
例)$(x-3y+1)(x+3y+1)$ → 並び替え
→ $(A-3y)(A+3y) = A^2 - 9y^2$
パターン3：$(A+a)^2$ 型 → 公式②
例)$(x+y-2)^2$
→ $(A-2)^2 = A^2 - 4A + 4$
「共通部分を見つけたら $A$ に置く」　これが最大のコツ！
あとはEF01〜EF03の公式を使うだけ。
📌 まとめ
  • 項が多い式は，共通部分を $A$ に置き換える
  • 手順：① $A$ と置く → ② 公式で展開 → ③ $A$ を戻して全部展開
  • パターン1：$(A+a)(A+b)$ → 公式①（たして・かける）
  • パターン2：$(A+a)(A-a)$ → 公式③（和と差の積）
    ※必要なら 並び替え
  • パターン3：$(A+a)^2$ → 公式②（平方公式）
  • 高認頻出！　$(a-2b+c)(a-2b-c)$ や $(a+b\pm n)(a+b\mp n)$ の問題は 毎年のように 出る

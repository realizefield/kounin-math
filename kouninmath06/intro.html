① 「たして・かける」の因数分解とは？
📌 EF01「たして・かける」展開の逆
EF01では，$(x+a)(x+b)$ を展開して $x^2 + (a+b)x + ab$ を作りました。
この単元では，逆に $x^2 + bx + c$ の形を $(x+\bigcirc)(x+\square)$ に分解 します。
📌 メインの考え方
$x^2 + bx + c = (x+\bigcirc)(x+\square)$
$\bigcirc + \square = b$，$\bigcirc \times \square = c$
例)$x^2 + 5x + 6$ を因数分解する
考え方
「たして $5$，かけて $6$」になる2つの数を探す
候補
かけて $6$ になるのは：$1 \times 6$，$2 \times 3$
条件チェック
たして $5$ になるのは $2$ と $3$
完成
$x^2 + 5x + 6 = (x+2)(x+3)$
$(x+2)(x+3)$
② 符号で 3つのパターン に分かれる
📌 $b$ と $c$ の符号で，2数の符号が決まる
$x^2 + bx + c$ の中で，$b$（$x$ の係数）と $c$（定数項）の符号 によって，
2つの数の符号の組み合わせが決まります。
高認では，この 3パターン をマスターすることが必須！
パターン1
$(x+\bigcirc)(x+\square)$　両方プラス
条件：$b > 0$，$c > 0$（両方プラス）
例)$x^2 + 5x + 6 = (x+2)(x+3)$
→ EF06-1 で詳しく学ぶ
パターン2
$(x-\bigcirc)(x-\square)$　両方マイナス
条件：$b < 0$，$c > 0$（係数マイナス・定数プラス）
例)$x^2 - 6x + 5 = (x-1)(x-5)$
→ EF06-2 で詳しく学ぶ
パターン3
$(x+\bigcirc)(x-\square)$　符号が逆
条件：$c < 0$（定数項マイナス）
例)$x^2 + 2x - 15 = (x+5)(x-3)$
→ EF06-3 で詳しく学ぶ
③ 3パターンの判別フロー
📌 まずは「定数項」の符号を見る
どのパターンかを見分けるには，$c$（定数項） の符号を見るのが 一番早い。
1
$c$（定数項）はプラス？
→ Yes：パターン1か2（同じ符号）
2
$c > 0$ なら $b$（係数）の符号を見る
　・$b > 0$ → パターン1（両プラス）
　・$b < 0$ → パターン2（両マイナス）
3
$c$（定数項）はマイナス？
→ Yes：パターン3（符号が逆）
💡 シンプルにまとめると
・定数項 $+$，係数 $+$ → 両プラス（パターン1）
・定数項 $+$，係数 $-$ → 両マイナス（パターン2）
・定数項 $-$ → 符号が逆（パターン3）
④ 例で見比べてみよう
3つの式を見分ける
パターン1
$x^2 + 7x + 12 = (x+3)(x+4)$　← 両プラス
パターン2
$x^2 - 7x + 12 = (x-3)(x-4)$　← 両マイナス
パターン3
$x^2 + x - 12 = (x+4)(x-3)$　← 符号が逆（定数項がマイナス）
3つの式は 定数項が同じ $12$ でも，符号によって 分解の仕方が違う！
だから「定数項の符号」と「係数の符号」を，最初によく見る必要がある。
⑤ かけて何になる？を考える
📌 まずは「かけて $c$」になる組み合わせを書き出す
たとえば $c = 12$ なら，かけて $12$ になる組み合わせは：
$1 \times 12$，$2 \times 6$，$3 \times 4$
その中で たして $b$ になる組 を選ぶ。
小さい数から書き出す方がラク！
$c$ が大きい場合（例：$36$，$48$など）は組み合わせが多いので，順序立てて 書き出すと見落としが減る。

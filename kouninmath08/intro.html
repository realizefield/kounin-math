① 共通因数とは？
📌 「みんなに共通する因子」をくくり出す
$3x + 6$ という式を見ると，$3x$ と $6$ には $3$ という共通の数 が含まれています：
$3x = 3 \times x$，$6 = 3 \times 2$
この共通の $3$ を 「共通因数」 と呼び，前にくくり出すことができる。
$3x + 6 = 3(x + 2)$　← これも因数分解！
📌 共通因数くくり出しの公式
$ax + ay = a(x + y)$
※ $a$ が共通因数
例)$2x + 4$ を因数分解する
観察
$2x = 2 \times x$，$4 = 2 \times 2$ → 共通因数は $2$
くくり出す
$2x + 4 = 2(x + 2)$
確認
展開すると $2(x+2) = 2x + 4$ ◎
$2(x + 2)$
② 文字も共通因数になる
📌 数だけでなく，文字も共通項として扱える
$x^2 + 3x$ という式を見ると，どちらの項にも $x$ が入っている。
$x^2 = x \times x$，$3x = 3 \times x$
だから $x$ をくくり出せる：$x^2 + 3x = x(x + 3)$
例)$x^2 + 5x$ を因数分解する
観察
$x^2 = x \times x$，$5x = x \times 5$ → 共通因数は $x$
くくり出す
$x^2 + 5x = x(x + 5)$
$x(x + 5)$
③ 数 × 文字の組み合わせ
📌 数と文字 両方が共通する場合
$6x^2 + 9x$ では，どちらの項にも $3$ と $x$ が共通する。
$6x^2 = 3 \times 2 \times x \times x$，$9x = 3 \times 3 \times x$
共通する $3$ と $x$ をまとめて $3x$ をくくり出す。
例)$6x^2 + 9x$ を因数分解する
数の共通
$6$ と $9$ の 最大公約数は $3$
文字の共通
両方に $x$ が入っている → 共通する文字は $x$
合わせる
共通因数は $3x$
くくり出す
$6x^2 + 9x = 3x(2x + 3)$
確認
$3x \times 2x = 6x^2$，$3x \times 3 = 9x$ ◎
$3x(2x + 3)$
💡 コツ：「最大公約数」と「共通文字」を見つける
① 数の 最大公約数 を見つける
② どちらの項にも入っている 共通文字 を見つける
③ 両方を合わせて 共通因数 としてくくり出す
④ 解き方の手順
1
各項を「数 × 文字」に分解
例：$6x^2 = 2 \times 3 \times x \times x$，$9x = 3 \times 3 \times x$
2
共通する部分を見つける
数：最大公約数（$3$）／文字：すべての項に入っている文字（$x$）
3
共通因数 × ( 残り ) の形にする
例：$3x \times (2x + 3) = 3x(2x + 3)$
4
展開して検算 すると安心！
⑤ 因数分解は 必ずまずコレ
📌 因数分解の最初の一手
どんな因数分解の問題でも，最初に共通因数があるかチェック するのが鉄則！
共通因数があれば 先にくくり出してから 他の方法（差の積，たして・かける，完全平方）を使う。
⚠️ 重要：因数分解の優先順位
① まず 共通因数（EF08）を確認
↓
② くくり出した残りに対して，他の方法を適用
↓
・$x^2 - a^2$ → 差の積（EF05）
・$x^2 + bx + c$ → たして・かける（EF06）
・$x^2 \pm 2ax + a^2$ → 完全平方（EF07）
⑥ 共通因数くくり出し → さらに因数分解
例)$2x^2 - 8$ を因数分解する
①共通因数
$2x^2 = 2 \times x^2$，$8 = 2 \times 4$ → 共通因数は $2$
くくり出す
$2x^2 - 8 = 2(x^2 - 4)$
②さらに
$x^2 - 4 = x^2 - 2^2$ → 差の積（EF05）
最終形
$2(x^2 - 4) = 2(x+2)(x-2)$
$2(x+2)(x-2)$
$2(x^2 - 4)$ で 止まらない こと！
さらに分解できるなら，「これ以上分解できない形」まで分ける のが因数分解。

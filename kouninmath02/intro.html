① 「2乗」の展開を考える
📌 $(x+3)^2$ ってどういう意味？
$(x+3)^2$ は $(x+3) \times (x+3)$ ということ。
つまり 同じものを2回かける。
EF01でやった「たして・かける」でも解けますが，もっと 速くて間違えない公式 があります。
確認）$(x+3)^2$ をふつうに計算してみる
2回かけ算
$(x+3)^2 = (x+3)(x+3)$
ばらす
$= x \times x + x \times 3 + 3 \times x + 3 \times 3$
計算
$= x^2 + 3x + 3x + 9$
同類項
$= x^2 + 6x + 9$
$(x+3)^2 = x^2 + 6x + 9$
② 平方公式①：$(x+a)^2$
📌 これがメイン公式その1！
$(x+a)^2 = x^2 + 2ax + a^2$
📌 各項の意味を理解しよう
$(x+a)^2$ を展開すると，3つの項ができます：
  • 1番目の項 $x^2$：前の項を 2乗
  • 2番目の項 $2ax$：前の項と後ろの項を かけて2倍
  • 3番目の項 $a^2$：後ろの項を 2乗
💡 覚え方：「前を2乗・前後をかけて2倍・後ろを2乗」
真ん中の項を 「前後をかけて2倍」 と覚えるのがコツ！
両端の $x^2$ と $a^2$ は，それぞれ「前」と「後ろ」を 2乗 するだけ。
例)$(x+4)^2$ を展開する
前を2乗
$x^2$
前後をかけて2倍
$x \times 4 \times 2 = 8x$
後ろを2乗
$4^2 = 16$
$(x+4)^2 = x^2 + 8x + 16$
③ 平方公式②：$(x-a)^2$
📌 これがメイン公式その2！
$(x-a)^2 = x^2 - 2ax + a^2$
📌 マイナスの場合の注意
後ろの項がマイナスの場合，真ん中の項だけマイナス になります。
最後の項 $a^2$ は 必ずプラス（マイナス × マイナス = プラス）！
$(x-a)^2$ の展開で，最後を $-a^2$ と書くのは 大きな間違い。
$(-a) \times (-a) = +a^2$ なので，必ずプラスになります！
例)$(x-5)^2$ を展開する
前を2乗
$x^2$
前後をかけて2倍
$x \times (-5) \times 2 = -10x$
後ろを2乗
$(-5)^2 = +25$　← プラス！
$(x-5)^2 = x^2 - 10x + 25$
④ 係数がついている場合
📌 $(3x+1)^2$ のようなケース
$x$ に係数がついていても，公式の使い方は同じ。
ただし 「前を2乗」する時は係数も含めて2乗 する点に注意！
たとえば $(3x)^2 = 3^2 \times x^2 = 9x^2$ となります。
例)$(3x+1)^2$ を展開する
前を2乗
$(3x)^2 = 9x^2$　← 係数も2乗！
前後をかけて2倍
$3x \times 1 \times 2 = 6x$
後ろを2乗
$1^2 = 1$
$(3x+1)^2 = 9x^2 + 6x + 1$
例)$(2x-5y)^2$ を展開する
前を2乗
$(2x)^2 = 4x^2$
前後をかけて2倍
$2x \times (-5y) \times 2 = -20xy$
後ろを2乗
$(-5y)^2 = 25y^2$　← プラス！
$(2x-5y)^2 = 4x^2 - 20xy + 25y^2$
⑤ 2つの公式を見比べよう
$(x+a)^2 = x^2 + 2ax + a^2$
真ん中の項：プラス $2ax$
最後の項：プラス $a^2$
$(x-a)^2 = x^2 - 2ax + a^2$
真ん中の項：マイナス $2ax$
最後の項：プラス $a^2$　←ここが重要！
マイナスがついても，最後の項は必ずプラスになることを覚えておこう！
📌 まとめ
  • $(x+a)^2 = x^2 + 2ax + a^2$
  • $(x-a)^2 = x^2 - 2ax + a^2$
  • 覚え方：「前を2乗・前後をかけて2倍・後ろを2乗」
  • 係数がある場合は 係数も含めて2乗（例：$(3x)^2 = 9x^2$）
  • $(x-a)^2$ でも最後の項は 必ずプラス！
  • 真ん中の項の符号だけが，後ろの項の符号で決まる

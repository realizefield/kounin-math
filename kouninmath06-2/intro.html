① パターン2の特徴
📌 定数項は「プラス」だが係数は「マイナス」
$x^2 - 6x + 5$ のように，
$x$ の係数が マイナス，定数項が プラス の式。
この形は，$(x - \bigcirc)(x - \square)$ の形 に分解できる。
つまり 2つの数は，両方マイナス。
📌 パターン2の形
$x^2 - bx + c \;\;\rightarrow\;\; (x-\bigcirc)(x-\square)$
※ $\bigcirc, \square$ はどちらも 正の数
② なぜ「両マイナス」になるの？
📌 符号のしくみを理解しよう
2つの数を $-A$ と $-B$（両方マイナス）とすると：
・たすと：$(-A) + (-B) = -(A+B)$ → マイナス（係数 $b$ がマイナスになる）
・かけると：$(-A) \times (-B) = +AB$ → プラス（定数項 $c$ がプラスになる）
だから 「係数マイナス・定数項プラス」 → 両方マイナス なのです！
例)$x^2 - 6x + 5$ を因数分解する
符号確認
定数項 $+5$（プラス），係数 $-6$（マイナス）→ 両マイナス
①かけて+5
$(-1) \times (-5) = +5$　← 負×負＝正
②たして-6
$(-1) + (-5) = -6$ ◎
③答え
$x^2 - 6x + 5 = (x-1)(x-5)$
$(x-1)(x-5)$
③ パターン1と比べてみよう
📌 定数項の符号は同じ，係数の符号で別れる
パターン1（両プラス）と パターン2（両マイナス）は，
どちらも 定数項がプラス なのは共通。
違いは 係数の符号！
パターン1（両プラス）
$x^2 + 6x + 5$
$= (x+1)(x+5)$
パターン2（両マイナス）
$x^2 - 6x + 5$
$= (x-1)(x-5)$
数の組み合わせ（$1$ と $5$）は 同じ！　ただ 符号が反対 になっただけ。
だから，パターン1で慣れていれば，パターン2もすぐにマスターできる！
④ 効率的な解き方
📌 パターン1と同じ要領で，最後に符号をマイナスにする
手順は パターン1とほぼ同じ：
① かけて $c$（プラス） になる正の数の組を書き出す
② たして $|b|$（係数の絶対値）になる組を選ぶ
③ 両方マイナスにして 答えを書く
例)$x^2 - 9x + 18$ を因数分解する
符号確認
定数項 $+18$，係数 $-9$ → 両マイナス
$1 \times 18$　たすと $19$
$2 \times 9$　たすと $11$
$3 \times 6$　たすと $\mathbf{9}$ ◎
他にはない
答え
$3$ と $6$ を 両方マイナスにして $(x-3)(x-6)$
$(x-3)(x-6)$
💡 コツ：「絶対値」で考える
係数 $-9$ はそのままでは扱いにくい。
でも「絶対値」で考えれば $9$ なので，
「かけて $18$，たして $9$」 と頭で計算すれば良い。
最後に答えを書くときに 両方マイナス をつけるだけ。
⑤ 注意点：負の数を考える
パターン1では 正の数だけ を考えればOKでしたが，
パターン2では 2つとも負の数 になるので，最後の符号付けに注意！
よくある間違い
×
$x^2 - 9x + 18 = (x+3)(x+6)$ → 展開すると $x^2 + 9x + 18$ になっちゃう
○
$x^2 - 9x + 18 = (x-3)(x-6)$ → 展開すると $x^2 - 9x + 18$ ◎
⑥ 大きい数の例
例)$x^2 - 11x + 30$ を因数分解する
かけて30
$1 \times 30$，$2 \times 15$，$3 \times 10$，$5 \times 6$
$1 \times 30$　たすと $31$
$2 \times 15$　たすと $17$
$3 \times 10$　たすと $13$
$5 \times 6$　たすと $\mathbf{11}$ ◎
答え
両方マイナスにして → $(x-5)(x-6)$
$(x-5)(x-6)$
📌 まとめ
  • パターン2：$x^2 - bx + c$（係数マイナス・定数項プラス）
  • 分解の形は $(x-\bigcirc)(x-\square)$（両方マイナス）
  • 手順：① かけて $c$（プラス）→ ② たして $|b|$ → ③ 両方マイナスにする
  • パターン1と 数の組み合わせは同じ，符号だけ反対
  • 「絶対値」で考えて，最後に符号をつけるとラク
  • $(-A) \times (-B) = +AB$（負×負＝正）の性質が鍵

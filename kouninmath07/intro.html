① 「完全平方」の因数分解とは？
📌 EF02「平方公式」展開の逆
EF02では，$(x+a)^2$ や $(x-a)^2$ を展開して $x^2 \pm 2ax + a^2$ を作りました。
この単元では，逆に $x^2 \pm 2ax + a^2$ の形を $(x \pm a)^2$ に分解 します。
これを「完全平方」（または「平方完成」）と言います。
📌 完全平方の公式
$x^2 + 2ax + a^2 = (x+a)^2$
$x^2 - 2ax + a^2 = (x-a)^2$
例)$x^2 + 6x + 9$ を因数分解する
観察
$9 = 3^2$，係数 $6 = 2 \times 3$ → $3$ の完全平方！
公式適用
$x^2 + 2 \times 3 \times x + 3^2 = (x+3)^2$
$(x+3)^2$
② 完全平方の見分け方（3つの条件）
📌 これが完全平方かどうか，どう見分ける？
式が $x^2 + bx + c$ の形のとき，次の 3つの条件 を満たすかチェックする：
1
定数項 $c$ がプラスで，しかも平方数
例：$c = 4 = 2^2$，$c = 9 = 3^2$，$c = 25 = 5^2$ など
2
係数 $b$ が「定数項のルート × 2」になっている
$c = a^2$ なら，$b = \pm 2a$ かどうかチェック
3
係数 $b$ の符号で $\pm$ を決める
$b > 0$ なら $(x+a)^2$，$b < 0$ なら $(x-a)^2$
💡 シンプルにまとめると
① 定数項 $c$ は ○の2乗 になっている？
② 係数 $b$ は ○の2倍 になっている？
YESなら → 完全平方公式が使える！
③ 完全平方かどうか判定してみよう
例1）$x^2 + 8x + 16$
①
定数項 $16 = 4^2$ ◎
②
係数 $8 = 2 \times 4$ ◎
→
完全平方！　$(x+4)^2$
$(x+4)^2$
例2）$x^2 - 10x + 25$
①
定数項 $25 = 5^2$ ◎
②
係数 $-10 = -(2 \times 5)$ ◎（マイナスがついた）
→
完全平方！　$(x-5)^2$
$(x-5)^2$
例3）$x^2 + 5x + 4$（完全平方ではない）
①
定数項 $4 = 2^2$ ◎
②
係数 $5$ は $2 \times 2 = 4$ と一致しない ×
→
完全平方ではない！　別パターンで因数分解
解
たして $5$，かけて $4$ → $(x+1)(x+4)$
定数項が平方数でも，係数の条件 も満たしてないと完全平方にならない！
両方の条件をチェックすること。

⑤ 高認頻出パターン
例)$x^2 + 12x + 36$ を因数分解する
観察
$36 = 6^2$，$12 = 2 \times 6$ → $6$ の完全平方
公式
$x^2 + 2 \times 6 \times x + 6^2 = (x+6)^2$
$(x+6)^2$
例)$x^2 - 14x + 49$ を因数分解する
観察
$49 = 7^2$，$-14 = -(2 \times 7)$ → $7$ の完全平方（マイナス）
公式
$x^2 - 2 \times 7 \times x + 7^2 = (x-7)^2$
$(x-7)^2$
⑥ 「たして・かける」でも解けるけど…
📌 完全平方は実はパターン1・2の特殊形
実は，$x^2 + 6x + 9$ も「たして $6$，かけて $9$」で解ける（$3$ と $3$）。
でも，同じ数が2回登場 するので $(x+3)(x+3) = (x+3)^2$ となる。
完全平方公式を使うと 一発で答えが出る のがメリット！
どちらの方法でも同じ答え
公式
$x^2 + 6x + 9 = (x+3)^2$（一発）
たして・かけて
たして $6$，かけて $9$ → $3$ と $3$ → $(x+3)(x+3) = (x+3)^2$
📌 まとめ
  • EF07は 完全平方の因数分解（EF02の逆操作）
  • 公式：$x^2 + 2ax + a^2 = (x+a)^2$，$x^2 - 2ax + a^2 = (x-a)^2$
  • 判定の 2つの条件：
  • 　・定数項 $c$ が 平方数（$a^2$）
  • 　・係数 $b$ が $\pm 2a$（$a$ の2倍）
  • 係数の符号で $(x+a)^2$ か $(x-a)^2$ かが決まる
  • 定数項の 平方数（$1, 4, 9, 16, 25, 36, 49, 64, 81, 100$）を覚えておく
  • 完全平方が使えれば 一発で答えが出る！

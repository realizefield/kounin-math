① 不思議な現象を見てみよう
📌 $(x+4)(x-4)$ をふつうに展開すると…
EF01の「ばらしてかける」でやってみると：
$(x+4)(x-4) = x \times x + x \times (-4) + 4 \times x + 4 \times (-4)$
$= x^2 - 4x + 4x - 16$
$= x^2 - 16$
あれ？真ん中の項が消えた！
確認）$(x+4)(x-4)$ を計算してみる
ばらす
$x \times x + x \times (-4) + 4 \times x + 4 \times (-4)$
計算
$= x^2 - 4x + 4x - 16$
注目
真ん中の $-4x$ と $+4x$ が 打ち消し合う！
結果
$= x^2 - 16$
$(x+4)(x-4) = x^2 - 16$
同じ数の足し算と引き算がペアになっているとき，真ん中の項が必ず消えます。これが「和と差の積」の特徴！
② 公式：和と差の積
📌 これがメイン公式！
$(x+a)(x-a) = x^2 - a^2$
📌 「和と差の積は，平方の差」
左の式 $(x+a)(x-a)$ は，「足したもの」と「引いたもの」のかけ算。
これを 「和と差の積」 と呼びます。
その答えはいつも $x^2 - a^2$（前の2乗 − 後ろの2乗） になります。
これを 「平方の差」 と呼びます。
💡 覚え方：「前を2乗，引く，後ろを2乗」
$(x+a)(x-a)$ を見たら，すかさず：
$x^2 - a^2$　← 前を2乗，後ろを2乗，間にマイナス！
真ん中の項を計算する手間が省けて，とても速い。
例)$(x+5)(x-5)$ を展開する
形を確認
$+5$ と $-5$ の 和と差の積
前を2乗
$x^2$
後ろを2乗
$5^2 = 25$
引く
$x^2 - 25$
$(x+5)(x-5) = x^2 - 25$
③ 順番が逆でも同じ
📌 $(x-a)(x+a)$ でも結果は同じ
$(x-3)(x+3)$ も，$(x+3)(x-3)$ と 同じ答え になります（かけ算の順番は関係ない）。
どちらも $x^2 - 9$ です。
例)$(x-7)(x+7)$ を展開する
形を確認
$-7$ と $+7$ の 和と差の積（順番が逆でもOK）
公式適用
$x^2 - 7^2$
$(x-7)(x+7) = x^2 - 49$
④ 係数がついている場合
📌 $(3x+1)(3x-1)$ のようなケース
$x$ に係数がついていても，公式は同じ。
ただし 「前を2乗」する時は係数も含めて2乗 する点に注意。
$(3x)^2 = 9x^2$ となります。
例)$(3x+1)(3x-1)$ を展開する
前を2乗
$(3x)^2 = 9x^2$　← 係数も2乗！
後ろを2乗
$1^2 = 1$
引く
$9x^2 - 1$
$(3x+1)(3x-1) = 9x^2 - 1$
⑤ 2変数の場合
📌 $(5x+2y)(5x-2y)$ のようなケース
$y$ がついていても，「和と差の積」の形なら公式が使えます。
$(5x)^2 - (2y)^2 = 25x^2 - 4y^2$
例)$(5x-2y)(5x+2y)$ を展開する
前を2乗
$(5x)^2 = 25x^2$
後ろを2乗
$(2y)^2 = 4y^2$
引く
$25x^2 - 4y^2$
$(5x-2y)(5x+2y) = 25x^2 - 4y^2$
⑥ 平方公式と見比べよう
$(x+a)(x-a) = x^2 - a^2$
異なる符号のかけ算
→ 答えは項が 2つだけ（真ん中なし）
→ 間は マイナス
$(x+a)^2 = x^2 + 2ax + a^2$（EF02）
同じ符号のかけ算（2乗）
→ 答えは項が 3つ（真ん中あり）
→ 間は プラス（または マイナス）
$(x+a)(x-a)$ と $(x+a)^2$ は 見た目が似ている が，まったく別の公式！
形をよく見て使い分けよう。
📌 まとめ
  • 公式：$(x+a)(x-a) = x^2 - a^2$
  • 覚え方：「前を2乗，引く，後ろを2乗」
  • 「和と差の積」は，必ず 「平方の差」になる
  • $(x-a)(x+a)$ も同じ答え（順番は関係ない）
  • 係数がある場合は 係数も含めて2乗（例：$(3x)^2 = 9x^2$）
  • 項が 2つだけ になるのが特徴（真ん中の項は消える）

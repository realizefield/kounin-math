定数項−　符号が逆

① パターン3の特徴
📌 定数項が「マイナス」
$x^2 + 2x - 15$ のように，
定数項が マイナス の式は，
$(x + \bigcirc)(x - \square)$ の形 — つまり 2数の符号が逆 になります。
パターン1・2と違って，係数の符号は プラスでもマイナスでもどちらでもOK。
📌 パターン3の形
$x^2 \pm bx - c \;\;\rightarrow\;\; (x+\bigcirc)(x-\square)$
※ 2数は 片方がプラス，もう片方がマイナス
② なぜ「符号が逆」になるの？
📌 符号のしくみを理解しよう
2つの数を $+A$ と $-B$（符号が逆）とすると：
・かけると：$(+A) \times (-B) = -AB$ → マイナス（定数項 $c$ がマイナスになる）
・たすと：$A + (-B) = A - B$ → 絶対値が大きい方の符号になる
だから 「定数項マイナス」 ならパターン3で，係数の符号はどちらもアリ。
例)$x^2 + 2x - 15$ を因数分解する
符号確認
定数項 $-15$（マイナス）→ パターン3（符号が逆）
①かけて-15
絶対値で $1 \times 15$ または $3 \times 5$
②差を考える
たして $+2$ → 差が $2$ になる組 → $3$ と $5$
③符号を決める
たして $+2$（プラス）→ 大きい方 $5$ がプラス → $+5$ と $-3$
④答え
$x^2 + 2x - 15 = (x+5)(x-3)$
$(x+5)(x-3)$
③ 「差」で考えるのがコツ
📌 パターン1・2との違い
パターン1・2では 2数の和 が係数になりましたが，
パターン3では2数の符号が違うので，2数の差 が係数になります。
だから 「かけて $|c|$，差が $|b|$」 となる組を探すのがポイント。
💡 解き方の手順
① かけて $|c|$（絶対値）になる正の数の組を書き出す
② その中から 差が $|b|$ になる組を選ぶ
③ 係数 $b$ の符号 に合わせて，どちらがプラスでどちらがマイナスか決める
⚠️ 符号の決め方
たして $b$（プラス）→ 絶対値の大きい数を プラス，小さい数を マイナス
たして $b$（マイナス）→ 絶対値の大きい数を マイナス，小さい数を プラス
（要するに，大きい方の数の符号 = 係数 $b$ の符号）
④ パターン1・2と比べてみよう
パターン1（両プラス）：定数項 +，係数 +
$x^2 + 8x + 15 = (x+3)(x+5)$
「たして $8$，かけて $+15$」
パターン2（両マイナス）：定数項 +，係数 −
$x^2 - 8x + 15 = (x-3)(x-5)$
「たして $-8$，かけて $+15$」
パターン3（符号が逆）：定数項 −
$x^2 + 2x - 15 = (x+5)(x-3)$
「たして $+2$，かけて $-15$」
パターン3では，2数の 差が係数 になる！
パターン1・2の「和」と違うので，最初は混乱しやすい。
⑤ 別パターン：係数がマイナスの場合
例)$x^2 - 4x - 12$ を因数分解する
符号確認
定数項 $-12$ → パターン3，係数 $-4$（マイナス）
①かけて12（絶対値）
$1 \times 12$，$2 \times 6$，$3 \times 4$
$1, 12$　差は $11$
$2, 6$　差は $\mathbf{4}$ ◎
$3, 4$　差は $1$
他にはない
③符号
係数が $-4$（マイナス）→ 大きい方 $6$ が マイナス，小さい方 $2$ が プラス
④答え
$+2$ と $-6$ → $(x+2)(x-6)$
$(x+2)(x-6)$
パターン3で 一番間違えやすい のは符号付け！
「差が大きい方は，係数 $b$ と 同じ符号」をしっかり覚えよう。
⑥ 大きい数の例
例)$x^2 + 5x - 24$ を因数分解する
かけて24
$1 \times 24$，$2 \times 12$，$3 \times 8$，$4 \times 6$
$1, 24$　差は $23$
$2, 12$　差は $10$
$3, 8$　差は $\mathbf{5}$ ◎
$4, 6$　差は $2$
符号
係数 $+5$ → 大きい方 $8$ が プラス，小さい方 $3$ が マイナス
答え
$+8$ と $-3$ → $(x+8)(x-3)$
$(x+8)(x-3)$
📌 まとめ
  • パターン3：定数項がマイナス（係数の符号はどちらでもOK）
  • 分解の形は $(x+\bigcirc)(x-\square)$（符号が逆）
  • 手順：① かけて $|c|$ → ② 差 が $|b|$ → ③ 係数の符号で大きい方の符号を決める
  • パターン1・2の「和」と違って，パターン3は「差」
  • 覚え方：「大きい方 = 係数と同じ符号」
  • これでパターン1〜3すべて完成！　高認の因数分解問題はバッチリ

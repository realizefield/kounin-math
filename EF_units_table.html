第1部 展開（Expansion）— 5単元
EF01
展開の基本（たして・かける）
展開のもっとも基本的なパターン。$(x+a)(x+b)$ を展開する。
$(x+a)(x+b) = x^2 + (a+b)x + ab$
「たして・かける」の基本
EF02
平方公式
$( \ )^2$ の形を展開する。$\pm$ で2種類。
$(x \pm a)^2 = x^2 \pm 2ax + a^2$
2乗の展開＝3項になる
EF03
和と差の積
符号だけが違う2つのかっこの積を展開する。
$(x+a)(x-a) = x^2 - a^2$
2項になる！
EF04-1
公式の組み合わせ展開
EF01〜EF03の3公式を見分けて使い分ける統合練習。
3公式の判別と適用
公式選択の判断力
EF04-2
展開の置き換え（A置換）
共通部分を $A$ に置き換えてから展開する。高認頻出。
例)$(x+y+a)(x+y+b)$
3公式 + A置換
第2部 因数分解（Factorization）— 8単元
EF05
因数分解① 差の積
EF03の「逆」。2乗の差を見つけて分解する。
$x^2 - a^2 = (x+a)(x-a)$
平方数を覚えよう
EF06
因数分解② たして・かける（総論）
$x^2 + bx + c$ の因数分解。3パターンの全体像を見る。
$x^2 + bx + c \rightarrow (x+\bigcirc)(x+\square)$
3パターン判別フロー
EF06-1
パターン1：両プラス
定数項＋・係数＋ の場合。EF06のサブ単元。
$(x+\bigcirc)(x+\square)$
かけて＞先・たして＞後
EF06-2
パターン2：両マイナス
定数項＋・係数− の場合。EF06のサブ単元。
$(x-\bigcirc)(x-\square)$
負×負＝正
EF06-3
パターン3：符号が逆
定数項− の場合。EF06のサブ単元。
$(x+\bigcirc)(x-\square)$
大きい方＝係数と同じ符号
EF07
因数分解③ 完全平方
EF02の「逆」。$( \ )^2$ の形に分解する。
$x^2 \pm 2ax + a^2 = (x \pm a)^2$
定数項＝$a^2$、係数＝$\pm 2a$
EF08
因数分解④ 共通因数
共通因数をくくり出す。因数分解の最初の一手。
$ax + ay = a(x+y)$
最大公約数 × 共通文字
EF09
因数分解⑤ 置き換え（A置換）
EF04-2の「因数分解版」。共通部分を $A$ に置換。
例)$(x+y)^2 + b(x+y) + c$
3パターン × A置換
📌 学習の流れ（推奨順）
① 展開の基礎を固める　EF01 → EF02 → EF03 → EF04-1
② 展開の応用　EF04-2（A置換）
③ 因数分解の基礎　EF05 → EF06（→ EF06-1 → EF06-2 → EF06-3）
④ 因数分解の応用　EF07 → EF08 → EF09

※ EF06-1〜3 は EF06 の中で必要に応じて深掘りする構成です。
🔄 展開と因数分解の対応関係
展開と因数分解は 同じ公式の「逆向き」 の操作です。

・EF02（平方公式の展開）　↔　EF07（完全平方の因数分解）
・EF03（和と差の積の展開）　↔　EF05（差の積の因数分解）
・EF01（たして・かけて 展開）　↔　EF06（たして・かけて 因数分解）
・EF04-2（展開の A置換）　↔　EF09（因数分解の A置換）

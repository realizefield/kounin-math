① これまで学んだ3つの公式
📌 EF01〜EF03の総まとめ
展開の公式には，大きく分けて 3つの型 があります。
どれもすぐに思い出せるように，まずは並べて見比べてみよう。
公式①（EF01）
「たして・かける」
$(x+a)(x+b) = x^2 + (a+b)x + ab$
📌 後ろの数が 違う ときに使う
公式②（EF02）
「前を2乗・前後をかけて2倍・後ろを2乗」
$(x \pm a)^2 = x^2 \pm 2ax + a^2$
📌 2乗の形になっているときに使う
公式③（EF03）
「前を2乗 − 後ろを2乗」
$(x+a)(x-a) = x^2 - a^2$
📌 後ろの数が 同じで符号だけ違う ときに使う
② 公式の見分け方フロー
📌 問題を見たら，順番にチェック！
どの公式を使うかは，問題の 形を見るだけ で決まります。
迷ったら，下の順番でチェックしてみよう。
1
$( \ )^2$ の形になっているか？
→ Yes：公式②（平方公式）
2
後ろの数の絶対値が同じで，符号だけ違うか？
例）$(x+5)(x-5)$
→ Yes：公式③（和と差の積）
3
それ以外（後ろの数が違う）
例）$(x+3)(x-7)$
→ 公式①（たして・かける）
💡 簡単な見分け方
・$( \ )^2$ → 公式②
・$(x+a)(x-a)$ で $a$ が同じ → 公式③
・それ以外 → 公式①
③ 例で確かめてみよう
例1）$(x+4)(x+2)$　→　どの公式？
チェック1
$( \ )^2$ ではない → 公式②ではない
チェック2
後ろは $+4$ と $+2$，絶対値が違う → 公式③ではない
判定
→ 公式①（たして・かける）
適用
たして $4+2=6$，かけて $4 \times 2=8$
$x^2 + 6x + 8$
例2）$(x-3)^2$　→　どの公式？
チェック1
$( \ )^2$ の形 → 公式②！
判定
→ 公式②（平方公式）
適用
前を2乗 $x^2$，前後をかけて2倍 $-6x$，後ろを2乗 $9$
$x^2 - 6x + 9$
例3）$(x+8)(x-8)$　→　どの公式？
チェック1
$( \ )^2$ ではない → 公式②ではない
チェック2
後ろは $+8$ と $-8$，絶対値が同じで符号が違う → 公式③！
判定
→ 公式③（和と差の積）
適用
前を2乗 $x^2$，後ろを2乗 $64$，引く
$x^2 - 64$
④ 紛らわしいパターンに注意
$(x+5)(x-5)$ と $(x+5)(x+5)$ と $(x+5)(x-3)$ — それぞれ違う公式を使う！
形をよく見て使い分けよう。
紛らわしい3問を比較
A) $(x+5)(x-5)$
同じ数の和と差 → 公式③ → $x^2 - 25$
B) $(x+5)(x+5)$
これは $(x+5)^2$ → 公式② → $x^2 + 10x + 25$
C) $(x+5)(x-3)$
後ろの数が違う → 公式① → $x^2 + 2x - 15$
形を見て公式を判断する → 答えがまったく違うので， 第一印象でしっかり見分ける ことが超重要！
⑤ 高認過去問パターン：$(a+b)(a-b+1)$
📌 3つの項がある場合
たまに，かっこの中に 3つの項 がある問題が出ます。
例)$(a+b)(a-b+1)$
こういう時は，ばらしてかける 基本に戻れば解けます。
（EF04-2では 「置き換え」 でもっと楽に解く方法を学びます）
例)$(a+b)(a-b+1)$ を展開する
ばらす
$a \times a + a \times (-b) + a \times 1 + b \times a + b \times (-b) + b \times 1$
計算
$= a^2 - ab + a + ab - b^2 + b$
同類項
$-ab$ と $+ab$ が消える
$a^2 - b^2 + a + b$
📌 まとめ
  • 展開の公式は 3種類：①たして・かける　②平方公式　③和と差の積
  • 見分け方：① $( \ )^2$ の形 → 公式②
  • 　　　　② 後ろの数が同じで符号だけ違う → 公式③
  • 　　　　③ それ以外 → 公式①
  • 形を 正しく見分ける ことが最大のポイント
  • 3つの項があるかっこは，ばらしてかける 基本に戻る
